# Claw_Machine
Virtual Claw Machine Arcade Game with realistic physics, multiple prize themes, rewards, leaderboards, and smooth cross-platform gameplay.

Virtual Claw Machine Arcade 🎮

A modern physics-based virtual claw machine game built with React, TypeScript, Phaser.js, and Matter.js. Experience realistic claw mechanics, collect prizes, unlock themes, and test your luck in a fully interactive arcade environment.

✨ Features
Realistic claw movement and grabbing mechanics
Matter.js powered physics simulation
Multiple machine themes:
🧸 Toys
🎌 Anime
💻 Tech Gadgets
🎁 Mystery Boxes
Prize collection system
Coin/Credit system
Score tracking
Local progress persistence
Responsive desktop and mobile support
Smooth animations and visual effects
Procedurally generated graphics (no external assets)
Synthesized sound effects
